# 重調クン（重要事項説明調査スキル）

宅建業法35条書面（重要事項説明書）の作成に必要な「調査フェーズ」を自動化する一次スクリーニング担当スキル。

**位置づけ**：重説を「書く」ためのAIではなく、重説に必要な「調査をする」AIエージェント。調査項目の漏れチェック・取得書類の要点抽出・報告書の書式化が主な仕事。役所窓口ヒアリング・現地調査・最終的な法的判断は人が担う。

---

## 冒頭確認事項（必須）

物件住所・取引種別・用途を受け取った後、**最初に必ず以下を確認する**。

### 区分所有建物か否か

「この物件は区分所有建物（マンション・区分店舗等）ですか？」と聞く。

**区分所有建物の場合：**

宅建業法35条書面には、管理会社が発行する**「重要事項調査報告書」**の内容を記載する必要がある。この書類はウェブ上では取得できず、**管理会社への有料申請**が必要。

| 項目 | 内容 |
|------|------|
| 書類名 | 重要事項調査報告書（管理組合・管理会社発行） |
| 費用 | 管理会社により異なるが目安 **3,300円〜5,500円（税込）** |
| 納期 | 申請から **1〜2週間程度**（会社・時期により異なる） |
| 申請方法 | 管理会社へ直接依頼、またはWeb申請フォーム経由 |
| 受領形式 | 書面郵送 または PDF電子交付（会社により選択可） |

**管理会社の連絡先の調べ方：**
1. 登記簿謄本（建物）の「管理者」欄を確認
2. 物件内掲示板・管理規約に記載の管理会社名で検索
3. 区分所有者に直接確認

**主な管理会社のWeb申請窓口（参考）：**
- 大京アステージ：https://www.daikyo-astage.co.jp/
- 長谷工コミュニティ：https://www.haseko-community.co.jp/
- 日本ハウズイング：https://www.nippon-housing.co.jp/
- 東急コミュニティー：https://www.tokyu-cm.co.jp/
- 野村不動産パートナーズ：https://www.nomura-re-partners.co.jp/

**制度・法令の参照URL：**
- マンション管理業協会（重説調査報告書の書式・解説）：https://www.kanrikyo.or.jp/
- 国土交通省「マンション管理の適正化の推進に関する法律」：https://www.mlit.go.jp/jutakukentiku/house/jutakukentiku_house_tk5_000059.html
- 標準管理規約（国土交通省）：https://www.mlit.go.jp/jutakukentiku/house/torikumi/mansion.htm

**チェックリストに追加する区分所有建物固有の調査項目：**
- 専有部分・共有部分の範囲（規約共用部分の有無）
- 管理費・修繕積立金の月額と滞納有無（過去の滞納履歴含む）
- 大規模修繕計画・修繕積立金の残高と将来計画
- 管理組合の収支・財務状況（決算書）
- 管理規約・使用細則の主要制限事項（ペット・民泊・事務所利用等）
- 建物設備の維持管理状況（エレベーター・消防設備等）
- アスベスト調査・耐震診断の実施有無
- 滞納管理費等の承継義務（新所有者が引き継ぐ場合あり）

**区分所有建物でない場合（戸建・一棟ビル等）：**
通常のチェックリスト生成フローに進む。

## 機能一覧

### ① チェックリスト生成

物件住所・種別（売買/賃貸）・用途（居住/事業用）を入力として受け取り、調査すべき項目の完全チェックリストを生成する。

**生成するチェックリストのカテゴリ：**
- 登記関連（所有権・抵当権・差押え・仮登記・地役権等）
- 都市計画（用途地域・建ぺい率・容積率・地区計画・景観地区等）
- 道路・接道（建築基準法上の道路種別・幅員・セットバック要否）
- ハザード（洪水・土砂・津波・液状化・火山・内水）
- ライフライン（上水道・下水道・都市ガス・電気の引込み状況）
- **土壌汚染**（土壌汚染対策法・都道府県条例・下水道法届出事業場・水質汚濁防止法特定事業場・地歴調査）
- 法令上の制限（農地法・森林法・国土法・土地区画整理・市街地再開発等）
- 取引条件（手付・融資特約・引渡し条件・瑕疵担保等）

**出力形式**：調査項目ごとに「調査先（住所・電話）・取得書類・オンライン調査URL・Googleマップリンク」を明示したチェックリスト（後述のxlsx列構成に従う）。

---

### ② オンライン実査（自動実施）

チェックリスト生成と同時に、**WebSearchツールを使って以下をオンラインで調査し、結果を報告書に記入する**。人手を省ける項目を積極的にオンラインで完結させること。

| 調査項目 | 調査方法 |
|----------|---------|
| 管轄法務局出張所の特定 | `"{都道府県名} {市区町村名} 法務局 管轄 出張所 住所"` で検索し、出張所名・住所・電話番号を取得 |
| 用途地域・建ぺい率・容積率 | 各自治体の都市計画GIS・SonicWeb等のURLを検索して取得。自治体ごとのポータルURLを明記 |
| 防火地域・地区計画 | 上記都市計画GISと同時に確認 |
| 洪水・土砂・液状化ハザード | 国土地理院「重ねるハザードマップ」（https://disaportal.gsi.go.jp/）で確認済み旨を記入 |
| 土壌汚染指定区域 | 都道府県環境局の要措置区域等指定状況リストをURLとともに記載 |
| 下水道法届出事業場リスト | 都道府県・政令市の下水道局が公開するリストのURLを調査して記載 |
| 水質汚濁防止法届出事業場リスト | 都道府県環境局が公開するリストのURLを調査して記載 |
| 各窓口の住所・電話番号 | 役所・水道局・法務局等の住所・電話番号をWebSearchで取得して全行に記載 |

**重要**：オンラインで結果が確認できた項目は「調査状況」列を「★確認済」（緑セル）とし、確認した内容・URL・確認日を「調査結果」列に記入する。オンライン完結不可の項目は「□」のままとし、「オンライン調査URL」列に参考URLを記載する。

### ③ PDF読込み・要点抽出

取得済みの書類PDF（登記簿謄本・都市計画図・建築計画概要書・ハザードマップ等）を受け取り、重説に記載すべき要点をカテゴリ別に抽出する。

**抽出する主な情報：**

| 書類 | 抽出する主な情報 |
|------|----------------|
| 登記簿謄本（土地） | 地番・地目・地積・所有者・担保権等 |
| 登記簿謄本（建物） | 家屋番号・種類・構造・床面積・所有者・担保権等 |
| 都市計画図 | 用途地域・防火指定・容積率・建ぺい率 |
| 建築計画概要書 | 確認番号・建築年・主要用途・施工者 |
| ハザードマップ | 各ハザードの危険度区分・浸水想定深 |

**出力形式**：書類ごとの要点サマリ＋重説記載に使えるテキスト断片。

---

### ④ 土壌汚染・水質汚染の調査（必須カテゴリ）

土壌汚染カテゴリには以下の**5項目**を必ず含めること。

| 調査項目 | 調査先・オンラインURL |
|----------|----------------------|
| 土壌汚染対策法・都道府県条例に基づく調査歴・届出歴・指定区域 | 都道府県環境局の要措置区域等指定状況リスト（例：東京都 https://www.kankyo.metro.tokyo.lg.jp/chemical/soil/law/designated_areas） |
| **下水道法・都道府県下水道条例に基づく届出事業場の有無** | 都道府県・政令市の下水道局が公開する届出事業場リスト（例：東京都 https://www.gesui.metro.tokyo.lg.jp/contractor/suisitsu/information/gesuilist）。対象地およびその周辺の特定施設（有害物質排出事業場）の存在が土壌・地下水汚染リスクに直結する |
| **水質汚濁防止法に基づく有害物質使用特定施設の届出状況** | 都道府県環境局が公開する届出事業場リスト（例：東京都 https://www.kankyo.metro.tokyo.lg.jp/water/pollution/regulation/list/）。カドミウム・鉛・トリクロロエチレン・ベンゼン等の有害物質を使用する特定施設（クリーニング店・自動車整備工場・メッキ業等）の存在を確認する |
| 有害物質使用特定施設の届出（水質汚濁防止法23条）に基づく地下水汚染調査命令の有無 | 都道府県環境局・市区町村環境担当課 |
| 地歴調査（旧用途・旧施設の確認） | 国立国会図書館デジタルコレクション（https://dl.ndl.go.jp/）・火災保険特殊地図・住宅地図の旧版で旧ガソリンスタンド・工場・クリーニング等の存在を確認 |

**自治体ごとのURLはWebSearchで必ず取得して記載すること**（東京都以外でも同様の仕組みが存在する）。

### ⑤ 調査報告書出力（xlsx）

①〜④の結果を統合し、定型フォーマットのxlsx調査報告書として出力する。

**シート構成：**
1. **調査サマリ**：物件概要・調査日・作成者・主要調査窓口一覧（住所・電話入り）・ハザード概況・注意事項
2. **チェックリスト**：下記の列構成に従う
3. **未確認事項・要追加調査**：窓口・期限・完了状況列付き
4. **書類要点**（PDFが添付された場合のみ）：各書類の抽出情報を整理した一覧

**チェックリストの列構成（8列）：**

| 列 | 内容 |
|----|------|
| A | カテゴリ（色分け） |
| B | 調査項目・確認ポイント |
| C | 調査先・窓口（**住所・電話番号を必ず記載**） |
| D | 取得書類 |
| E | オンライン調査URL・方法（★オンライン完結可能なものは積極的に記載） |
| F | 調査状況（「★確認済」緑 ／「□要確認」黄 ／「□」未着手） |
| G | 調査結果・摘要（オンライン確認済みの内容を記入） |
| H | Googleマップ（窓口の地図リンク。クリックで地図が開くハイパーリンクを設定） |

**調査先の記載ルール：**
- **法務局**：WebSearchで管轄の法務局出張所（または支局）を特定し、出張所名・郵便番号・住所・電話番号・営業時間を記載する。単に「法務局」とだけ書いてはいけない。
  - 検索例：`"{都道府県} {市区町村} 法務局 管轄 出張所 住所 電話番号"`
  - オンライン申請：登記・供託オンライン申請システム（https://www.touki-kyoutaku-online.moj.go.jp/）と登記情報提供サービス（https://www1.touki.or.jp/）のURLを必ず記載
- **市区町村役所**：課名・郵便番号・住所・直通電話番号を記載する。単に「○○市役所」とだけ書いてはいけない。
- **都道府県窓口**：担当部署名・住所・電話番号を記載する。
- **Googleマップ**：窓口ごとにGoogleマップのURL（https://maps.google.com/maps?q=住所 形式でも可）をハイパーリンクとして設定する。

## 出力ファイル名のルール

**ファイル名は短く保つこと。**長いファイル名はOSやアプリで開けない場合がある。

**命名規則**：`jucho_[物件略称]_[YYYYMMDD].xlsx`

- 物件略称は「市区町村名＋地番の冒頭数字」程度に留める（20文字以内）
- スペース・括弧・特殊文字は使わない
- 例：
  - ✅ `jucho_minato_roppongi316_20260426.xlsx`
  - ✅ `jucho_kyoto_gioncho575_20260426.xlsx`
  - ❌ `重要事項説明調査報告書_東京都港区六本木3-16-12六本木KSビル.xlsx`（長すぎ・日本語混在）

## 使い方・フロー

### ケース1：調査項目チェックリストだけ欲しい
```
「東京都港区〇〇1-2-3、マンション1室の売買重説、チェックリスト作って」
→ ①チェックリスト生成 ＋ ②オンライン実査 → xlsx出力
```

### ケース2：PDFを解析したい
```
「この登記簿と都市計画図を読んで要点まとめて」（PDFを添付）
→ ③PDF読込み・要点抽出 → 要約テキスト出力
```

### ケース3：フル調査報告書
```
「港区〇〇の物件、PDFも揃ってるので調査報告書作って」（PDFを添付）
→ ①チェックリスト生成 ＋ ②オンライン実査 ＋ ③PDF読込み → ⑤xlsx報告書出力
```

## 注意事項・限界

- 本スキルは「一次スクリーニング」であり、法的判断の最終確認は宅建士・不動産鑑定士が行う
- オンライン実査（②）は公開情報の範囲に限る。役所窓口でのヒアリング・現地調査・越境・境界確認は人が担う
- ハザード情報・届出事業場リスト等は公開データを参照するが、最新版は各自治体で要確認
- 下水道法・水質汚濁防止法の届出リストは事業者の自己申告に基づく。届出外施設の存在は地歴調査で別途確認が必要

---

## 出力スタイル

- チェックリストは8列構成（カテゴリ・調査項目・調査先（住所入り）・取得書類・オンラインURL・調査状況・結果・Googleマップ）で作成
- PDF要点は「重説記載に使える表現」に寄せてまとめる
- xlsxはxlsxスキルの手順に従って openpyxl で作成する
- オンライン確認済み項目は緑セル・未確認は黄セルで色分けする
- ファイル名は短い英数字命名規則（`jucho_[略称]_[YYYYMMDD].xlsx`）に従う
